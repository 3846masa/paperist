はじめに

推奨環境

Paperist では、以下のソフトウェアが必要です。

（これらのインストール方法は、付録 | 推奨環境のインストールを参照してください。）

  • Git
  • Node.js
    • バージョン 7.6.0 以上が必要です
  • LaTeX
    • TeXLive 2017 以上を推奨します
    • 以下のコレクションが必要です
      • collection-basic
      • collection-latex
      • collection-latexrecommended
      • collection-latexextra
      • collection-fontsrecommended
      • collection-langjapanese
      • latexmk

インストール

コマンドプロンプト もしくは ターミナルから、 以下のコマンドを実行してください。

npm install --global @paperist/cli

試してみる

まずは、執筆フォルダを作りましょう。

paperist init ./new-paper
cd ./new-paper

執筆フォルダには、 すでにサンプルの文章が入っています。

サンプルの文章を PDF にするには、 以下のコマンドを実行してください。

paperist build

build.pdf が生成されれば、 インストールは無事に完了しています。
