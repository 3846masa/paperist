# Paperist

![Paperist](./assets/paperist-logo.png)

**ようこそ Paperist へ！**

Paperist は、
**Markdown で論文や原稿を書く**ための環境セットです。

Node.js と LaTeX 環境があれば、
簡単に原稿を書き始めることができます。

さあ、[さっそく始めましょう](./getting-started/README.md)。
