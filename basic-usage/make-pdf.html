PDF を作成する

PDF を作成する

コマンドライン もしくは ターミナルで、 以下のコマンドを実行すると PDF が作成されます。

paperist build

また、 ファイルの変更を監視しつつ、 リアルタイムに PDF を作成することもできます。

以下のコマンドを実行すると、 ファイルの監視が始まります。

paperist watch

注意点

標準的な PDF ビュアーで開いているときには、 新しい PDF データで上書きができません。

（PDF をブラウザで開いている場合には、 この現象は発生しません。）

PDF を生成するときは、 PDF ビュアーを閉じて実行してください。
